Etiquetas comunes en HTML:

  • <html>: Define el documento HTML.
    Ejemplo: <html>...</html>
  • <head>: Contiene metadatos y enlaces.
    Ejemplo: <head><title>Mi página</title></head>
  • <body>: Contiene el contenido visible.
    Ejemplo: <body>Hola mundo</body>
  • <header>: Encabezado de la página o sección.
    Ejemplo: <header>Bienvenido</header>
  • <nav>: Navegación.
    Ejemplo: <nav><a href="#">Inicio</a></nav>
  • <main>: Contenido principal.
    Ejemplo: <main>Contenido</main>
  • <footer>: Pie de página.
    Ejemplo: <footer>Copyright</footer>
  • <h1>-<h6>: Encabezados.
    Ejemplo: <h1>Título</h1>
  • <p>: Párrafo.
    Ejemplo: <p>Texto</p>
  • <span>: Texto en línea.
    Ejemplo: <span>Rojo</span>
  • <div>: Contenedor genérico.
    Ejemplo: <div>Bloque</div>
  • <ul>: Lista desordenada.
    Ejemplo: <ul><li>Item</li></ul>
  • <ol>: Lista ordenada.
    Ejemplo: <ol><li>Item</li></ol>
  • <li>: Elemento de lista.
    Ejemplo: <li>Elemento</li>
  • <dl>: Lista de definiciones.
    Ejemplo: <dl><dt>Término</dt><dd>Definición</dd></dl>
  • <a>: Enlace.
    Ejemplo: <a href="https://devf.com">DevF</a>
  • <img>: Imagen.
    Ejemplo: <img src="foto.jpg" alt="Foto">
  • <figure>: Contenedor de imagen o gráfico.
    Ejemplo: <figure><img src="img.png"><figcaption>Descripción</figcaption></figure>
  • <table>: Tabla.
    Ejemplo: <table><tr><td>Dato</td></tr></table>
  • <form>: Formulario.
    Ejemplo: <form><input type="text"></form>
  • <input>: Campo de entrada.
    Ejemplo: <input type="text">
  • <label>: Etiqueta para campo.
    Ejemplo: <label for="nombre">Nombre</label>
  • <select>: Menú desplegable.
    Ejemplo: <select><option>Opción</option></select>
  • <textarea>: Área de texto.
    Ejemplo: <textarea>Texto</textarea>
  • <button>: Botón.
    Ejemplo: <button>Enviar</button>
  • <article>: Artículo independiente.
    Ejemplo: <article>Noticia</article>
  • <section>: Sección.
    Ejemplo: <section>Parte</section>
  • <aside>: Contenido lateral.
    Ejemplo: <aside>Publicidad</aside>
  • <details>: Información expandible.
    Ejemplo: <details><summary>Ver más</summary>Texto oculto</details>
  • <audio>: Reproductor de audio.
    Ejemplo: <audio src="audio.mp3" controls></audio>
  • <video>: Reproductor de video.
    Ejemplo: <video src="video.mp4" controls></video>
  • <script>: Código JavaScript.
    Ejemplo: <script>alert('Hola')</script>
  • <style>: Estilos CSS.
    Ejemplo: <style>body { color: red; }</style>
  • <link>: Enlace a recursos externos.
    Ejemplo: <link rel="stylesheet" href="style.css">
  • <meta>: Metadatos.
    Ejemplo: <meta charset="UTF-8">
