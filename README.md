
# Portafolio de Proyectos de Programación

## ¿Cómo ejecutarlo?

1. Descarga o clona este repositorio en tu computadora.
2. Abre la carpeta del proyecto.
3. Haz doble clic en el archivo `index.html` o ábrelo con tu navegador favorito.

¡Listo! Podrás ver tu portafolio funcionando de inmediato, sin necesidad de instalar nada extra.

---

Si quieres personalizarlo, edita los archivos `index.html` y `style.css` según tus necesidades.
